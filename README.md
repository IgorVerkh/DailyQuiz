## Технологии
- **UI**: Jetpack Compose
- **Навигация**: Compose Navigation
- **Работа с сетью**: Retrofit + Kotlinx Serialization
- **Кэш**: Room
- **DI**: Koin

## Сделано
- Фильтры и прохождение викторины
- Сохранение результатов в БД
- Работа с сетью + фильтры
- Таймер
